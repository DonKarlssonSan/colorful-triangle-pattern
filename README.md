# About
This is a demo of my Delaunay triangulation library [delaunay.js](https://github.com/DonKarlssonSan/Delaunay.js).

Try it out here: https://donkarlssonsan.github.io/colorful-triangle-pattern/

I'm using GitHub Pages to publish it.


# Run locally
Install the dependencies:
```
npm install
```

This will use parcel to run a local dev server:

```
npm start
```

Look at the terminal output for the URL, normally http://localhost:1234.
